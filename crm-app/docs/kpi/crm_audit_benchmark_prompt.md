## สิ่งที่ต้องวิเคราะห์ (ทุกข้อ)

### 1. ตัวเลขดิบ (Raw Numbers)
- จำนวนข้อความรวม
- จำนวนแชทที่ดูแล
- การกระจายเวลาตอบ (แยกเป็น: <5 นาที, 5-30 นาที, 30-60 นาที, 1-4 ชั่วโมง, >4 ชั่วโมง)
- ใครส่งข้อความสุดท้าย (Admin vs Customer)

### 2. Dropout Funnel Analysis
วิเคราะห์ว่าลูกค้าหายไปในกระบวนการไหน:
- หายหลังถามราคา?
- หายหลังได้รับข้อมูล (Info Overload)?
- หายเพราะตอบช้า?
- หายหลังถูกปฏิเสธ?
- หายด้วยเหตุอื่น?
อ้างอิง Conversation ID และข้อความจริงเป็นหลักฐาน

### 3. แพทเทิร์นการสื่อสาร
- ความเป็นหุ่นยนต์ (Robotic Pattern Score 1-10)
- Template Dependency
- ภาษา: สะกดผิด, ระดับภาษา, การใช้ Emoji
- Personalization: เรียกชื่อลูกค้าหรือไม่

### 4. Marketing Psychology Scorecard
ให้เกรด A-F สำหรับแต่ละหลักการ:
- Reciprocity (ให้สิ่งที่เกินความคาดหมาย)
- Urgency/Scarcity (สร้างความเร่งด่วน)
- Social Proof (อ้างอิงรีวิว/ผลงาน)
- Rapport Building (สร้างสายสัมพันธ์)
- Follow-up (ตามลูกค้าที่เงียบหาย)
- Anchoring (เทคนิคราคา)
- Consistency Binding (มัดจำความรู้สึก)

### 5. 🔥 การรับรู้อารมณ์และเจตนาลูกค้า (Emotional & Intent Detection)
**สำคัญมาก** — วิเคราะห์ว่าแอดมินสามารถ "อ่าน" ความรู้สึกและเจตนาที่ซ่อนอยู่ในข้อความลูกค้าได้ดีเพียงใด:
- เมื่อลูกค้ากังวล → แอดมินตอบด้วยความเห็นอกเห็นใจหรือแค่ตอบ Fact?
- เมื่อลูกค้าให้ข้อมูลโดยไม่ต้องถาม → แอดมินจับ Buying Signal ได้หรือไม่?
- เมื่อลูกค้าพูด "ไม่สะดวก" → แอดมินตีความว่า "ปฏิเสธ" หรือ "ต้องการทางเลือก"?
- เมื่อลูกค้าตอบ "สนใจ" สั้นๆ → แอดมินเข้าใจว่าเป็น Impulse ที่ต้องจับทันทีหรือไม่?
ระบุตัวอย่างจากแชทจริงทุกข้อ พร้อมบอกว่า "AI อ่านอารมณ์ลูกค้าได้ว่าอย่างไร" vs "แอดมินตอบอย่างไร" vs "ควรตอบอย่างไร"

### 6. จุดแข็ง vs จุดอ่อน
จัดลำดับความรุนแรง: 🔴 วิกฤต / 🟡 ปานกลาง / 🟢 เล็กน้อย

### 7. ข้อเสนอแนะเชิงปฏิบัติ (Actionable)
แบ่งเป็น: ต้องแก้ทันที / ควรปรับปรุง / ปรับเพื่อความสมบูรณ์แบบ
ให้ตัวอย่าง script ใหม่ที่ควรใช้

### 8. บทสรุป (Final Verdict)
- ประเมินมูลค่าที่สูญเสียไป (ถ้ามี)
- สรุปว่าแอดมินคนนี้คือ "ประเภทไหน" (เช่น Information Dispatcher, Sales Consultant, Hybrid)

## กฎเหล็ก
1. **ห้ามอวย ห้ามเข้าข้าง** — วิจารณ์ตรงไปตรงมาเหมือนที่ปรึกษาที่ถูกจ้างมาตรวจสอบ
2. **ทุกข้อวิจารณ์ต้องมีหลักฐาน** — อ้างอิงข้อความจริงจากแชทเสมอ
3. **ใช้หลักจิตวิทยาการตลาด** — อ้างหลักทฤษฎีเมื่อเหมาะสม (เช่น Cognitive Load Theory, FEEL-FELT-FOUND, Anchoring)
4. **ภาษาไทย** — เขียนรายงานเป็นภาษาไทย
5. **ลงชื่อ** — ระบุชื่อโมเดลและเวอร์ชั่นของคุณท้ายรายงาน
