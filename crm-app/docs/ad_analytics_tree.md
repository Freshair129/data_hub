## สรุปสูตรการคำนวณ (Metric Logic)

|ตัวชี้วัด (Metric)|ความหมาย|สูตรการคำนวณ|ความสำคัญ|
|:---|:---|:---|:---|
|**CPM**|ต้นทุนต่อการเห็น 1,000 ครั้ง|`(Spend / Impressions) * 1,000`|วัดความถูก/แพงของกลุ่มเป้าหมาย (Traffic Cost)|
|**CTR**|อัตราการคลิกต่อการมองเห็น|`(Clicks / Impressions) * 100`|วัดความน่าสนใจของรูปภาพ/วิดีโอ (Creative Quality)|
|**CPC**|ราคาเฉลี่ยต่อ 1 คลิก|`Spend / Clicks`|ผลลัพธ์ของ CPM และ CTR รวมกัน|
|**CPL**|ต้นทุนเฉลี่ยต่อ 1 รายชื่อลูกค้า|`Spend / Leads`|วัดความคุ้มค่าในการหาลูกค้ารายใหม่|
|**ROAS**|ผลตอบแทนจากงบโฆษณา|`Revenue / Spend`|ตัวชี้วัดที่ "บอส" ให้ความสำคัญที่สุด (ความคุ้มค่าเชิงธุรกิจ)|

## ลำดับการไหลของข้อมูล (Data Aggregation Flow)

1.  **ระดับล่างสุด (Daily/Hourly Metrics):** ระบบดึงข้อมูลดิบ (Spend, Imp, Click) รายวันและรายชั่วโมงจาก FB API มาเก็บลง `AdDailyMetric` และ `AdHourlyMetric`
2.  **ระดับโฆษณา (Ad Level):** ระบบรวมผลจาก Metric รายวันเข้าสู่ตาราง `Ad` เพื่อแสดงผลรายชิ้นงาน
3.  **ระดับกลุ่มเป้าหมาย (AdSet Level):** ยอดรวมของ Ads ทั้งหมดภายใต้ AdSet นั้นๆ จะถูกนำมาคำนวณหาค่าเฉลี่ยและยอดรวม
4.  **ระดับแคมเปญ (Campaign Level):** ยอดบนสุดที่สรุปทุกอย่างในแคมเปญ เพื่อให้ฝ่ายบริหารดูภาพรวม ROAS ของคอร์สเรียนนั้นๆ
